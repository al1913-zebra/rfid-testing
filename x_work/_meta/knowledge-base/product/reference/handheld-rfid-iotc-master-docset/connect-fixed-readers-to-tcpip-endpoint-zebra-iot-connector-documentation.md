-   Connect Fixed Readers to TCP/IP Endpoint

___

This section serves to guide the user to connect Zebra fixed readers `FX9600`, `FX7500` and `ATR7000` with `TCP/IP` endpoint.

Important

prerequisite:

-   The firmware version on `FX9600` and `FX7500` and `ATR7000` should be >= `3.24.X`
    

Warning

Reader uses `user configured` port to listen to connections from a `tcpip client`. Port should be open for this tutorial to work.
