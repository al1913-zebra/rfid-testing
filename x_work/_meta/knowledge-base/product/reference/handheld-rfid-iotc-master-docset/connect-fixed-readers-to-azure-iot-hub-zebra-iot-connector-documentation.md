This section serves to guide the user to connect Zebra fixed readers `FX9600`, `FX7500` and `ATR7000` with `Azure IoT Hub` service.

Important

prerequisite:

-   `FX9600` and `FX7500` readers firmware version should be >= `3.10.X`
    
-   `ATR7000` reader firmware version should be >= `3.17.X`
    

Warning

Reader uses `8883` port to connect with `Azure IoT Hub`. Port should be open for this tutorial to work.
