
Manage TLS/SSL certificates for secure device communication and authentication. You can install, remove, and inspect certificates for CA, client, MQTT, and WiFi, ensuring only trusted devices and servers can connect.

| Command | Description |
|---|---|
| [get_installed_certificate](#op-get-installed-certificate) | List all certificates currently on the device |
| [install_certificate](#op-install-certificate) | Upload a certificate (CA, client, MQTT, WiFi) to the device |
| [delete_certificate](#op-delete-certificate) | Remove an installed certificate by name |
