
Check the device’s current state and identity without making changes. These read-only commands help you monitor health, verify device details, and ensure regulatory compliance for safe and reliable operation.

| Command | Returns |
|---|---|
| [get_status](#op-get-status) | Real-time health — battery level, temperature, radio activity, power source, NTP sync, terminal connection |
| [get_version](#op-get-version) | Device identity — model, serial number, SKU, firmware versions (main, scanner, radio, IoTC) |
| [get_current_region](#op-get-current-region) | Radio regulation — country code, frequency channels, power limits, frequency hopping, LBT |
