
Configure where the device sends data by setting broker connection parameters such as URL, security, and topics. This ensures secure and reliable data flow to your chosen endpoints.

| Command | Description |
|---|---|
| [get_endpoint_config](#op-get-endpoint-config) | Retrieve current endpoint configuration |
| [config_endpoint](#op-config-endpoint) | Configure broker URL, port, protocol, credentials, topics, and TLS settings |
