| Command | Description |
|---|---|
| [get_operating_mode](#op-get-operating-mode) | Retrieve current operating mode settings and parameters |
| [set_operating_mode](#op-set-operating-mode) | Configure RFID operating mode — transmit power, session, tag access operations |
