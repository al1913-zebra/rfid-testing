# Introduction[](https://zebradevs.github.io/rfid-ziotc-docs/about/index.html/#introduction "Link to this heading")

This documentation provides information about the Zebra IoT Connector feature in the FX Series and ATR7000 readers. It explains the system’s overall architecture, deployment modes, and configuration of different deployment modes.

Important

If you have a problem with your equipment, contact Zebra Global Customer Support for your region. Contact information is available at: [zebra.com/support](https://www.zebra.com/us/en/support-downloads/rfid.html).

## Supported Readers[](https://zebradevs.github.io/rfid-ziotc-docs/about/index.html#supported-readers "Link to this heading")

-   FX7500
    
-   FX9600
    
-   ATR7000
    

## Chapter Descriptions[](https://zebradevs.github.io/rfid-ziotc-docs/about/index.html#chapter-descriptions "Link to this heading")

Topics covered in this documentation are as follows:

> -   [Overview](https://zebradevs.github.io/rfid-ziotc-docs/introduction/intro/index.html#overview) provides a summary on Zebra IoT Connector, the cloud-based management, control, device health and RFID tag data collection for the fixed RFID readers.
>     
> -   [Deployment Modes](https://zebradevs.github.io/rfid-ziotc-docs/introduction/deployment_modes/index.html#deployment-modes) provides information on the different modes of deployment.
>     
> -   [Operation Modes](https://zebradevs.github.io/rfid-ziotc-docs/introduction/operating_modes/index.html#operation-modes) provides information on configuring the radio to different modes of operation.
>     
> -   [Setting Up Zebra IoT Connector](https://zebradevs.github.io/rfid-ziotc-docs/setupziotc/index.html#ziotc-setup) provides information on setting up a reader for hybrid deployment with MQTT and HTTP POST endpoints.
>     
> -   How To Guides provides information on pure cloud deployment with services offered by Amazon Web Services and Azure.
>     
> -   API Reference provides Open API documentation of Zebra IoT Connector REST APIs.
>     
> -   Schemas provides Open API documentation of various schemas used by Zebra IoT Connector services.
>

## Service Information[](https://zebradevs.github.io/rfid-ziotc-docs/about/index.html#service-information "Link to this heading")

If you have a problem with your equipment, contact Zebra Global Customer Support for your region. Contact information is available at: zebra.com/support. When contacting support, please have the following information available: [zebra.com/support-Downloads](https://www.zebra.com/ap/en/support-downloads.html).

> -   Serial number of the unit
>     
> -   Model number or product name
>     
> -   Software/firmware type or version number
>     

Zebra responds to calls by email, telephone or fax within the time limits set forth in support agreements.

If your problem cannot be solved by Zebra Customer Support, you may need to return your equipment for servicing and will be given specific directions. Zebra is not responsible for any damages incurred during shipment if the approved shipping container is not used. Shipping the units improperly can possibly void the warranty.

If you purchased your Zebra business product from a Zebra business partner, contact that business partner for support.
