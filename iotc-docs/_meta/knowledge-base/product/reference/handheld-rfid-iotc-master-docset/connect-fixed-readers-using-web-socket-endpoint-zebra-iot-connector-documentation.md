-   Connect Fixed Readers using Web Socket Endpoint

___

This section serves to guide the user to connect to Zebra fixed readers `FX9600`, `FX7500` and `ATR7000` using `Web Socket`.

Important

prerequisite:

-   The firmware version on `FX9600` and `FX7500` and `ATR7000` should be >= `3.24.X`
