There are few settings in `123RFID Desktop` can be exported via `Save Config` page on 123RFID Desktop and can be saved as a JSON file.

> # Save Config[](https://zebradevs.github.io/rfid-ziotc-docs/migration/123RFID_Desktop/save.html/#save-config "Link to this heading")
> 
> ![123RFID Desktop User Applications](https://zebradevs.github.io/rfid-ziotc-docs/_images/SaveConfig.png)

Configuration, which will be exported are Antennas DWell Time, Antenna Enable/Disable, RF Mode, Singulation, State Aware parameters, Tag Population, Report Filter, TAG META DATA and Antenna Power.

The exported JSON file content can be copied and set via `Operating Mode` in Zebra IoT Connector. As like below.

> ![Zebra IoT Connector](https://zebradevs.github.io/rfid-ziotc-docs/_images/Custom1.png)
