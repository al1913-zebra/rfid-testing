Zebra IoT Connector for Handheld Readers (v2)

Download OpenAPI specification:

Zebra IoT Connector API Documentation for Handheld Readers (RFD40/90) supported via Wi-Fi and Ethernet Interfaces. The interface is built on a web-friendly protocol MQTT and JSON-formatted data structures that can be leveraged by applications running on the cloud or on-premise to manage and control readers.

Control

Control commands are structured APIs that enable users to control Handheld RFID device operations. These commands facilitate actions such as starting or stopping inventory scans, retrieving or setting operating modes, and configuring post-filters, ensuring precise and efficient control over the device's functionality.
