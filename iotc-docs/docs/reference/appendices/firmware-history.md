:::caution[Not an authoritative changelog]
The versions, dates, and feature deltas below are **indicative** and have **no source in the IOTC API schema** (the schema carries no firmware changelog). They are a reading aid, not a release record. Confirm the exact version string, release date, and per-version deltas against Zebra's official firmware release notes for your specific model before you plan an update.
:::
