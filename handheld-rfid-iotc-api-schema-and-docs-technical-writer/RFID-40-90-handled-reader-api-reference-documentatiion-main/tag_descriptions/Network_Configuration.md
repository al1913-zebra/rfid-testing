
Manage both WiFi and Ethernet connections for flexible device networking. View status, configure access points, and remove saved profiles to keep your device securely connected at all times.

| Command | Description |
|---|---|
| [get_eth](#op-get-eth) | Retrieve Ethernet status including link state, speed, IP address, security |
| [get_wifi](#op-get-wifi) | Retrieve WiFi status including connected SSID, IP, signal strength, security type |
| [set_wifi](#op-set-wifi) | Configure WiFi access point, security (WPA2/WPA3/Enterprise), and IP settings |
| [delete_wifi_profile](#op-delete-wifi-profile) | Remove a saved WiFi profile by SSID |
