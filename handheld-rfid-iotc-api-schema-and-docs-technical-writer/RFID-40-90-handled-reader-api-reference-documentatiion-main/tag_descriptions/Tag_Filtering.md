
Control which tags are reported after inventory scans by applying flexible post-read filtering rules. Match or exclude tags based on EPC patterns, masks, or other criteria to focus on the data that matters most.

| Command | Description |
|---|---|
| [get_post_filter](#op-get-post-filter) | Retrieve current tag filter configuration |
| [set_post_filter](#op-set-post-filter) | Define filter rules to match/exclude tags by EPC pattern, mask, or criteria |
